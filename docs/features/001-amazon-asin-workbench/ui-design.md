# 001：初级网站 UI 设计

## 决策记录

- 设计方向：**A · 证据优先工作台**，已由用户在可视化草图中点选；根据随后提供的多页面参考，先升级为 Precision Glass Console，后按用户反馈改为 **Summer Glass Operations**。不复制参考站的布局或配色。
- 用户：中文亚马逊运营；主要任务是研究单个 ASIN，而不是批量看大盘。
- 页面形态：单页应用内的多模块控制台。默认是“今日控制台”，可切换产品研究、Listing、广告策略、创意资产和利润护栏；本期以桌面端为验收范围。
- 数据状态：所有内容是内置演示数据，绝不使用“实时”“已抓取”等误导用语。

## 视觉系统

| 项目 | 规则 |
| --- | --- |
| 风格 | 清透、白亮、安静的夏日玻璃工作台；冰水远景只提供冷感氛围，运营信息永远在前景。不是营销落地页，也不是满屏产品海报。 |
| 画布 | 冰白 `#F4FBFA`、雾蓝 `#E6F3FA`、极淡薄荷和水波光；以低对比、经过淡化处理的冰水照片作为全屏远景。 |
| 玻璃层 | `rgba(255,255,255,.68-.84)` + `backdrop-filter: blur(22px) saturate(1.15)` + 浅青灰细边框。卡片透，但文本区域有足够白色遮罩以保持读数清晰。 |
| 强调色 | 海盐蓝 `#4D9DCC` 用于主要操作；薄荷 `#32B89E` 表示完成；日光黄 `#EDB950` 表示阶段/注意；珊瑚 `#E97875` 表示风险。颜色永远配合图标和文字。 |
| 正文 | 墨青 `#173D42`；次级 `#55727A`；边框 `rgba(58,114,124,.17)`。 |
| 字体 | 中文优先系统无衬线；英文/数字用 `ui-monospace` 做证据编号、预算和标签，形成精确感。 |
| 间距 | 4、8、12、16、24、32、48 px。 |
| 圆角/阴影 | 16–22 px 主卡、12 px 内部控件；像漂在水汽上的极浅蓝灰阴影，不使用黑色重投影。 |
| 图标 | 单色线性 SVG（Lucide），不得使用 emoji 代替功能图标。 |

## 布局与内容优先级

### 桌面（>= 1024 px）

1. 248 px 半透明白玻璃侧栏：产品身份与六个运营模块；当前模块是水蓝玻璃胶囊，非当前模块仅在 hover 时微亮。
2. 顶部状态条：当前产品、阶段、更新时间、内置演示数据徽章和“启动演示分析”。
3. 默认控制台：左侧显示新品推进路径与今日三件事；右侧显示“Launch D-12”状态、完成度、风险/机会和待决策事项。
4. 产品研究：ASIN 输入、演示进度、产品档案、证据与 Brief；遵循先证据后结论。
5. Listing、广告、创意、利润模块：分别呈现文案框架、投放结构、创意缺口、利润护栏；每一屏的第一行必须回答“下一步做什么”。
6. 所有内容卡采用 16–22 px 圆角、低对比细描边和缓慢的内容入场错峰；冰水背景不得降低任何正文对比度。

### 小屏（< 1024 px）

- 当前代码可保留基础自适应兜底，但不作为本期验收范围或视觉优化目标。
- 后续若启用移动端，需要新建任务，重审导航、信息优先级、触控目标与图片画廊策略。

## 关键交互与状态

| 组件 | 默认行为 | 状态规则 |
| --- | --- | --- |
| ASIN 输入 | 初始填入演示 ASIN，允许替换 | 空或格式错误时按钮禁用并展示中文说明；焦点有海盐蓝描边。 |
| 启动演示分析 | 在本地依次推进六个演示步骤 | 点击后显示“正在准备内置演示档案”，完成后聚焦到产品研究的档案。 |
| 步骤时间线 | 展示完成、进行中、待处理 | 颜色 + 图标 + 文案共同表达，不只靠颜色。 |
| 证据标签 | 如 `证据 E-003` | 点击时高亮相应档案字段或图片资产；键盘可触达。 |
| 草稿复制 | 复制标题/五点/关键词建议 | 成功提示“已复制草稿”；失败提供可选中文说明。 |
| 导出 | 禁用 | 明示“真实数据版开放”，不制造无响应按钮。 |
| 即将开放导航 | 可见但非主操作 | 点击显示轻提示，不跳转到空页面。 |
| 模块切换 | 默认进入控制台 | 使用 180–240ms 的 opacity/translateY 淡入，键盘切换不添加多余位移动画。 |
| 今日三件事 | 每项有目的、依赖和完成态 | 勾选仅改变本地演示状态；完成时显示轻量薄荷反馈，不播放庆祝烟花。 |

## 无障碍与质量要求

- 语义 HTML：`nav`、`main`、`section`、`button`、`label`、`article`；所有输入有可见 label。
- 所有交互可键盘操作，有明显 `:focus-visible`；支持 `prefers-reduced-motion`。
- 文本与背景达到至少 4.5:1 对比度；“风险”除金色外含图标与文字。
- 状态由 ARIA live 区域播报；图片有清楚的演示性替代文本。
- 本期桌面断点：1024、1440 px；不允许桌面页面横向溢出。小屏断点仅作非验收兜底。
- 动效使用 `transform` 和 `opacity`，入场 180–260ms、错峰 40ms；水光背景只做极慢的透明度呼吸，不做人物运动；所有大幅位移在 `prefers-reduced-motion` 下关闭。

## 夏日背景资产

- 背景资产是用户在本对话提供的本地静态图片 `public/assets/ice-water-bottles.jpg`；仅做远景氛围，不表示用户、店铺、Amazon 或运营数据。
- 选择依据：冰蓝、薄荷绿、透明水瓶和冷藏雾感能提供夏日清凉感，同时没有人物妆容或角色扮演的语义干扰。
- 使用规则：背景须经低饱和、提亮、轻微柔焦与白色叠层处理；商品文字不得作为页面信息或功能文案。正式发布前由项目方确认其对图片和包装文字的使用权。

## 初级版本屏幕清单

实现一个可切换的本地运营控制台，含控制台、产品研究、Listing、广告策略、创意资产、利润护栏六个演示视图。真实失败、历史研究、多 ASIN 对比、登录、设置和导出文件均留给后续功能包。
